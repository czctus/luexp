### luexp - API Framework for Roblox

**luexp** is an API framework designed for Roblox, allowing developers to easily create RESTful APIs within Roblox games. With **luexp**, you can quickly set up endpoints to handle HTTP requests and build powerful APIs for your Roblox projects.

- **Easy-to-Use**: Create APIs with simple, expressive syntax.
- **HTTP Method Support**: Handle GET, POST, PUT, DELETE, and other HTTP methods.
- **Route Handling**: Define routes and their corresponding actions.
- **Middleware and Routers** (Coming Soon): Additional features for advanced API development.

### Server Installation

**You will require node.js for this**
**Note: By default you can use api.perox.dev servers instead of hosting your own.**

#### Install Dependencies

`UUID`
```bash
$ npm install uuid
```

`Express`
```bash
$ npm install express
```

`Axios`
```bash
$ npm install axios
```

#### Install Server

1. Get the most recent Installation [Here](https://github.com/czctus/luexp/releases)
2. Move the server.js anywhere
3. Run ``cd path.to.server`` & ``node server.js`` in a terminal
4. Double check the server is running on `http://localhost:3001` by going to `http://localhost:3001/s/ver`

#### Install Client

1. Get the most recent Installation [Here](https://github.com/czctus/luexp/releases)
2. Drag and Drop the downloaded `RBXM` or `RBXMX` to Roblox Studio
3. Move the **luexp** module anywhere
4. Run one of the tests located in `luexp/tests` to check if its working.
